# Junhee Lee Portfolio

이준희의 공개 개발자 포트폴리오입니다. 복잡한 고객 환경에서 동작하는 백엔드 기능과 다시 확인할 수 있는 AI 품질 기준을 중심으로, 공개 검토를 통과한 사례만 보여줍니다.

- Live (Vercel): <https://jun-portfolio-ten.vercel.app/>
- Fallback (GitHub Pages): <https://junhee742.github.io/jun-portfolio/>
- Repository: <https://github.com/junhee742/jun-portfolio>

## 기술 구성

- Astro 7
- MDX content collections
- Noto Sans KR Variable
- Phosphor Icons
- Vercel 배포

## 로컬 실행

Node.js 22.12 이상이 필요합니다.

```sh
npm install
npm run dev
```

## 검증

```sh
npm run verify
```

## 콘텐츠 수정

프로젝트 상세 내용은 `src/content/work/`에 있습니다. 새 사례를 공개하기 전에는 반드시 원본 Jun Wiki에서 근거와 공개 안전성 검토를 완료해야 합니다.

이 저장소에는 고객명, 내부 제품명, 저장소·이슈·서버·DB 식별 정보, 비공개 근거 ID를 추가하지 않습니다.
